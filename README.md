# Microprofile demo
A microprofile sample using WildFly Swarm implementation.

To run:

* `mvn clean wildfly-swarm:run`
* `java -jar target/microprofile-demo.jar`

###The endpoints

* **GET** `/speakers`
* **GET** `/sessions`
* **GET**`/config`
* **GET**`/health`
* **GET**`/metric-demo`
* **GET**`/metrics`
